## 🧠 What is this?

This is my personal portfolio — a hand-crafted, pixel-obsessed, over-engineered shrine to everything I've built, broken, and rebuilt. I'm an **AI/ML Engineer from India** building **agentic AI**, **LLM inference systems**, and **multimodal models** for **speech**, **vision**, and **memory**.

> *"It started as a simple HTML page. Then I added a halftone blob canvas. Then a custom theme toggle. Then grain textures. Then I spent 4 hours adjusting a margin by 0.5rem. You know how it goes."*

**No frameworks. No templates. Just vibes, vanilla code, and questionable amounts of CSS.**

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/Komalpreet2809/Komal-s-Portfolio.git

# Navigate into the project
cd Komal-s-Portfolio

# Serve it (pick your poison)
```
